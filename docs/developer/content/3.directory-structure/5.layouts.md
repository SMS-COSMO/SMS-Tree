# Layouts 布局

::alert{type="success"}
关于 Layouts 的更多信息，请参阅 [Nuxt 文档](https://nuxt.com/docs/guide/directory-structure/layouts)。
::

```
.
└── layouts/
    ├── admin.vue
    └── default.vue
```

## `default`

默认布局，包含顶部导航栏。

## `admin`

管理布局，在管理页面使用。
