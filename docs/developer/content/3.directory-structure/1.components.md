# Components 组件

::alert{type="success"}
关于 Composables 的更多信息，请参阅 [Nuxt 文档](https://nuxt.com/docs/guide/directory-structure/components)。
::

`component` /kəmˈpəʊ.nənt/ n. 组件

顾名思义，该文件夹里包含了构建UI时候使用的组件。

```
.
└── components/
    ├── attachment # 附件相关
    ├── class # 班级相关
    ├── group # 小组相关
    ├── layouts # 布局相关
    ├── paper # 论文相关
    ├── user # 用户相关
    └── utils # 工具组件
```
